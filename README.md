# Deploy on Render
- This project is ready for Render using `render.yaml`.
- Render will install dependencies from `requirements.txt` and start the app with `gunicorn main:app`.

## Steps
1. Push this repository to GitHub.
2. In Render, click **New +** -> **Blueprint**.
3. Select your GitHub repo and confirm deployment.
4. Render reads `render.yaml` and creates the web service.
5. After deploy completes, open the generated Render URL.

## Environment variables
- `SECRET_KEY`: auto-generated by Render via `render.yaml` (you can override it in service settings).
- `PYTHON_VERSION`: set to `3.9.19`.
